# Assets — Visual Brand Files

Store all logos, thumbnails, banners, and other visual assets here.

---

## Folder Structure

```
/logos/           → Logo files (PNG, SVG, etc.)
/thumbnails/      → YouTube thumbnail files (1280x720px)
/banners/         → Channel banner (2560x1440px)
/other/           → Misc assets (textures, fonts, etc.)
```

---

## File Naming Convention

### Logos:
- `CYFI_Logo_Primary_[Color].png`
- `CYFI_Logo_Icon_[Color].png`
- Example: `CYFI_Logo_Primary_Black.png`

### Thumbnails:
- `CYFI_Mix[XX]_[ThemeName]_thumb.png`
- Example: `CYFI_Mix01_WhenYouLetGo_thumb.png`

### Banners:
- `CYFI_Banner_YouTube_[Version].png`
- Example: `CYFI_Banner_YouTube_v1.png`

---

## Design Guidelines (Keep It Simple)

### Logo:
- **Don't overthink this.** Use what you have or go text-based.
- Lock your decision after Mix 3 (not before)
- Avoid: Complex gradients, too many colors, tiny details (won't scale)

### Thumbnails (Most Important):
- **Size**: 1280x720px (YouTube standard)
- **Text**: Large, readable on mobile (50-60pt minimum)
- **Colors**: High contrast (dark bg + light text or vice versa)
- **Emotion**: The text should convey the feeling (match your title)
- **Consistency**: Use the same font + layout style across all mixes

**Template approach** (recommended):
1. Pick one background style (gradient, texture, photo)
2. Pick one font (bold, sans-serif, readable)
3. Use the same layout for every mix (only change text)
4. This creates visual consistency = brand recognition

### Channel Banner:
- **Size**: 2560x1440px (safe zone: 1546x423px)
- **Content**: Channel name + tagline ("Trip hop mixes that land")
- **Test**: Check how it looks on desktop + mobile + TV
- **Priority**: Low (use a simple design, don't spend >30 min on this)

## Tools (Free Options)

- **Canva** (free): Templates, easy to use
- **Figma** (free): More control, steeper learning curve
- **Photopea** (free): Browser-based Photoshop alternative
- **Remove.bg** (free): Remove image backgrounds
- **Unsplash/Pexels** (free): Stock photos for backgrounds

## Rules

1. **Don't let assets block shipping.** Text-based thumbnails work fine.
2. **Consistency > perfection.** Same style across mixes > "perfect" one-offs.
3. **Lock and move on.** After Mix 3, stop iterating on logo/style.
4. **Test on mobile.** 70%+ of viewers are on phones—design for that.

---

## Thumbnail Checklist (Use Before Uploading)

- [ ] Size: 1280x720px
- [ ] Text readable on phone screen (test by shrinking to 300px wide)
- [ ] High contrast (text pops against background)
- [ ] File size <2MB (YouTube limit)
- [ ] Saved as PNG or JPG
- [ ] Filename follows convention
- [ ] Conveys emotion (matches title/vibe)
