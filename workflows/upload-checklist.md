# YouTube Upload Checklist

Use this checklist for **every upload** to ensure consistency and avoid mistakes.

---

## Pre-Upload (Before Opening YouTube Studio)

- [ ] Final mix file exported (MP3 or WAV, 320kbps minimum)
- [ ] File named clearly: `CYFI_Mix01_[ThemeName]_Final.mp3`
- [ ] Thumbnail created and exported (1280x720 px, PNG or JPG)
- [ ] Title finalized (from template)
- [ ] Description finalized (from template)
- [ ] Tracklist formatted and timestamped
- [ ] Pinned comment drafted (email + Discord CTA)

---

## Upload Settings (YouTube Studio)

### Basic Info
- [ ] **Title**: Copy from template (check character count <100)
- [ ] **Description**: Paste full description (include tracklist, CTAs, credits)
- [ ] **Thumbnail**: Upload custom thumbnail (not auto-generated)
- [ ] **Playlist**: Add to "Can You Feel It â€” Mixes" playlist (create if first upload)

### Tags
- [ ] Add 10-15 relevant tags (see tag bank below)
- [ ] Include: `trip hop mix`, `lofi mix`, `emotional mix`, `late night music`, `[your channel name]`

### Visibility
- [ ] Set to **Unlisted** first (to review before going live)
- [ ] Review embedded player: thumbnail displays correctly, audio syncs
- [ ] Change to **Public** when ready

### Advanced Settings
- [ ] **Category**: Music
- [ ] **Comments**: Enabled (hold potentially inappropriate for review)
- [ ] **Age restriction**: None (unless explicit lyrics)
- [ ] **Allow embedding**: Yes
- [ ] **Publish to subscriptions feed**: Yes
- [ ] **License**: Standard YouTube License

---

## Post-Upload (Within 5 Minutes of Publishing)

- [ ] **Pin your CTA comment** (email signup + Discord invite)
- [ ] Add video to any relevant playlists
- [ ] Copy video URL and save in `/content/mix-[number]-metadata.md`
- [ ] Share to YouTube Community tab (if enabled): "New mix live ðŸŽ§ [Theme description]"

---

## Distribution (Within 1 Hour of Publishing)

- [ ] Post to 2 relevant subreddits (r/triphop, r/lofihiphop, r/chillmusic, etc.)
  - Use format: `[Trip Hop/Lofi] Title | 25min mix`
  - Include 1-sentence description of vibe
- [ ] Share to Instagram (static post or story with link in bio)
- [ ] Optional: Share in music Discord servers (2-3 max, no spam)
- [ ] Update your landing page if this is your first/featured mix

---

## Follow-Up (24-48 Hours After Publishing)

- [ ] Check YouTube Analytics: views, watch time, traffic sources
- [ ] Respond to any comments within 24 hours
- [ ] Note any copyright claims in `/content/mix-[number]-notes.md` (for future reference)
- [ ] If any tracks get muted/blocked, document which ones
